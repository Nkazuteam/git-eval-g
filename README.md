## お題: PS1（プロンプト）をカスタマイズしよう

ターミナルを開くと表示されるプロンプト（`$` の左側）を自分好みに変えてみましょう。

### やること

1. このリポジトリを clone する
2. `setup-prompt.sh` というファイルを作成する
3. そのスクリプトの中で `PS1` を設定する
4. ブランチを切って作業する（例: `git switch -c feature/my-prompt`）
5. commit して push する
6. GitHub 上で PR を作成する

### setup-prompt.sh の例

```bash
#!/bin/bash
# プロンプトにユーザー名とカレントディレクトリを表示する例
PS1="\u@\h:\w\$ "
```

よく使う PS1 のエスケープ:
- `\u` — ユーザー名
- `\h` — ホスト名
- `\w` — カレントディレクトリ（フルパス）
- `\W` — カレントディレクトリ（フォルダ名のみ）
- `\t` — 現在時刻
- `\n` — 改行

自由にカスタマイズして、自分だけのプロンプトを作ってみてください。

### 評価ポイント（100 点満点）

| チェック項目 | 配点 |
|---|---|
| `setup-prompt.sh` が存在する | 20 |
| スクリプトの bash 構文が正しい | 20 |
| `PS1=` が設定されている | 20 |
| CRLF 改行が混入していない | 20 |
| コミットが 2 つ以上ある | 10 |
| コミットメッセージが意味のある内容 | 10 |

### 提出方法

push するだけ！ GitHub Actions が自動で評価して Discord に結果を通知します。

### GitHub Secrets の設定

リポジトリの Settings → Secrets and variables → Actions に以下を追加してください:

| Secret 名 | 値 |
|---|---|
| `GIT_EVAL_URL` | 管理者から通知された Webhook URL |
| `GIT_EVAL_SECRET` | 管理者から通知された Secret キー |
